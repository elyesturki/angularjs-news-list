# AngularJS News list API + Filter List + Local Storage when connexion if OFF

## Clone

```sh
git clone git remote add origin https://github.com/elyesturki/angularjs-news-list.git
```

## Install

```sh
npm install
```

## Usage

```sh
npm start
```
